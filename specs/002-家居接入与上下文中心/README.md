# Spec 002 - 家居接入与上下文中心

状态：Draft（Spec 已补全，前端原型已落地，后端上下文能力待实现）

## 目标摘要

本 Spec 承接家庭底座之后的第二条主线：

1. 打通 `Home Assistant` 设备接入与基础动作执行
2. 建立在家状态事件、成员在家快照与房间占用缓存骨架
3. 提供统一的家庭上下文总览查询入口
4. 提供管理台仪表盘与配置界面，便于管理员理解与纠偏系统判断

## 当前文件

- `requirements.md`：业务范围、需求与验收标准
- `design.md`：模块设计、接口、数据模型、错误处理与测试策略
- `tasks.md`：任务拆分、依赖关系与阶段检查点
- `docs/20260309-家居上下文中心-前端原型说明-v0.1.md`：当前阶段前端原型说明

## 当前阶段说明

当前已先落一个**可演示的管理台原型页**，入口为：

- `apps/admin-web/src/pages/ContextCenterPage.tsx`
- 路由：`/context-center`

这个页面先把“家庭状态、成员状态、关键信息仪表盘、配置界面”做出来，配置暂以浏览器本地持久化承接；后续再按本 Spec 的接口设计替换为后端 `context overview / context configs` 能力。
