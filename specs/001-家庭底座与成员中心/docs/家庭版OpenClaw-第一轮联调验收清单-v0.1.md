# 家庭版 OpenClaw 第一轮联调验收清单 v0.1

## 1. 验收目标

本清单用于完成 `specs/001-家庭底座与成员中心/tasks.md` 中的 `T027`：

- 后端 + 前端 + HA 同步完整联调

本轮验收范围仅覆盖当前 Spec 已实现内容：

- 家庭底座
- 成员中心
- 成员关系
- 成员偏好与权限
- 房间与设备
- Home Assistant 设备同步
- 审计日志

不在本轮范围：

- 问答
- 长期记忆
- 语音
- 复杂场景编排

---

## 2. 环境检查

- [ ] Python `3.11` 可用
- [ ] Node.js 与 npm 可用
- [ ] `apps/api-server/.env` 已配置

## 5. 推荐验收步骤

建议按照如下顺序执行：

1. 启动后端服务
2. 启动前端服务
3. 检查 `/` 与 `/api/v1/healthz`
4. 创建 1 个家庭
5. 创建至少 2 个成员
6. 创建 1 条成员关系
7. 为 1 个成员保存偏好
8. 为 1 个成员保存权限
9. 创建 1 个房间
10. 触发 1 次 HA 设备同步
11. 更新 1 个设备的房间归属
12. 查询审计日志核对关键动作

---

## 6. 通过标准

满足以下条件即可判定 `T027` 通过：

- 后端服务启动正常
- 前端管理台启动正常
- SQLite 与迁移正常
- 家庭、成员、成员关系、偏好、权限、房间、设备接口形成闭环
- Home Assistant 同步成功执行
- 审计日志覆盖关键写操作
- 本 Spec 范围内未出现阻断性错误
