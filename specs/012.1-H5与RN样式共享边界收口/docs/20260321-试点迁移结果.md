## 现在的边界是什么

### 可以共享

- token
- 语义颜色和组件语义
- 页面布局模式
- 桌面/触屏的结构级降级规则

### 不可以共享

- `scss/css` 源文件
- `StyleSheet.create(...)`
- `hover`
- `matchMedia`
- `SafeArea`
- `elevation`
- 浏览器专属定位和滚动条细节

## 守卫和验证

- 样式守卫已经补进共享边界规则：

  3. 最后才改页面
