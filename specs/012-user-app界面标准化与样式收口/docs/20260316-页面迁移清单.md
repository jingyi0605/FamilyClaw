1. 哪些页面或公共块已经吃到共享层

3. 哪些还明确是历史债，不要假装已经收口

## 已建立的新基线

- 新的公共页头默认用 `PageHeader`
- 新的区块容器默认用 `PageSection`
- 新的卡片默认用 `UiCard`
- 新的空态默认用 `EmptyStateCard`
- 设置类开关默认用 `ToggleSwitch`
- 登录和简单表单默认用 `FormField` + `UiInput` + `UiButton`

## 后续继续推进时的顺序

1. 先清 `settings/channel-access` 和 `settings/components/*` 里重复出现的配置卡、明细卡、状态块
2. 再拆 `setup/index.h5.tsx` 的表单块和步骤块
3. 然后处理 `assistant` 的对话动作卡
4. 最后再处理 `home` 的仪表盘专有卡片
