# 机器人 PR 审核清单

## 先看什么

1. 对应收录 Issue
2. 自动校验结果
3. 插件源码仓库 README
4. 生成出来的 `entry.json`

## 必查项

- 插件源码仓库地址是真实可访问的
- `manifest.json` 和市场条目里的 `plugin_id`、权限、风险等级能对上
- README 不是摆设，至少能让人看懂插件干什么、怎么验证
- `requirements.txt` 和插件包根目录真实存在
- 自动生成的摘要和分类没有明显胡来
- 维护者信息不是空的

## 可以人工调整的地方

- `summary`
- `categories`
- `maintainers`

## 不能为了省事跳过的地方

- 权限声明
- 风险等级
- 插件仓库可信度
- 是否值得进入正式市场
