# 任务清单 - 第三方插件仪表盘卡片开放（人话版）

状态：Draft

## 这份文档是干什么的

这份任务清单的目的很简单：把“第三方插件首页卡片开放”拆成能执行、能回写、能验收的步骤。

这里不搞假大空路线图，只回答几件事：

- 先收口什么协议
- 再补什么后端数据和接口
- 然后怎么接 `user-app`
- 最后怎么验证这玩意不是纸上谈兵

## 状态说明

- `TODO`：还没开始
- `IN_PROGRESS`：正在做
- `BLOCKED`：被外部问题卡住
- `IN_REVIEW`：已经有结果，等复核
- `DONE`：已经完成，并且已经回写状态
- `CANCELLED`：取消，不做了，但要写原因

规则：

- 只有 `状态：DONE` 的任务才能勾选成 `[x]`
- `BLOCKED` 必须写清楚卡在哪里
- `CANCELLED` 必须写清楚为什么不做
- 每做完一个任务，必须立刻更新这里
