## 你现在最适合看哪一份

- 如果你还不清楚 FamilyClaw 能做什么、自己应该从哪一步开始，请先看 [快速开始](/快速开始/文档总览)。
- 如果你现在最关心的是“先把系统装起来”，直接进入 [安装部署](/安装部署/概览)。
- 如果你已经完成安装，接下来想知道每个页面怎么用、该怎么配置，请看 [使用手册](/使用指南/首次登录与初始化)。
- 如果你需要改代码、接接口、写插件，或者排查实现问题，请看 [开发手册](/开发文档/环境准备)。

## 这套文档会帮你解决什么

- 不知道先装什么、怎么选部署方式时，你可以先从安装文档里挑最适合自己的路径。
- 不清楚登录后每个页面是干什么的，可以直接从使用手册按页面逐步查看。
- 担心中途踩坑时，文档会优先把关键前提、常见误区和容易漏掉的步骤写清楚。
- 如果你是开发者，开发文档会直接告诉你入口、边界和应该遵守的规则，避免乱改。

## 文档分区一眼看懂

- `快速开始`：适合第一次接触 FamilyClaw 的人，先帮你建立整体认识，再带你进入下一步。
- `安装部署`：适合要把系统跑起来的人，覆盖 Docker、源码、NAS、Ubuntu、Windows 等常见方式。
- `使用手册`：适合已经开始使用的人，重点解释页面功能、操作流程和常见配置。
- `开发手册`：适合开发和联调场景，集中放环境准备、接口、插件和规范说明。
- `沟通交流`：放官网和社群入口，方便你继续获取更新和支持。
