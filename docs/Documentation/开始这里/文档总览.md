# 文档总览

::: info
这页是入口页，只负责告诉读者“先看什么、去哪里找”，不承载具体安装、使用或开发细节。
:::

## 文档怎么分

- `开始这里`：给第一次接触项目的人，先看全局，再决定往哪个方向走。
- `安装部署`：解决环境、安装、配置和部署问题。
- `使用指南`：解决普通使用者的操作路径问题。
- `开发指南`：解决开发者接手、调试、扩展和提交流程问题。
- `运维与排障`：解决日志、故障、升级和回滚问题。
- `参考资料`：放配置表、接口表、术语表这类查阅型内容。
- `存量资料`：保留当前仓库里已经存在的规范和旧文档入口。

## 建议阅读路径

### 我是第一次接触项目

1. 先看 [完整文档目录与写作计划](./完整文档目录与写作计划.md)
2. 再看 [快速开始](./快速开始.md)
3. 按角色进入安装、使用或开发章节

### 我要补正文档内容

1. 先看 [文档编写模板说明](../文档规范/文档编写模板说明.md)
2. 再看 [文档图片与排版约定](../文档规范/文档图片与排版约定.md)
3. 最后在对应模板页里填正文
