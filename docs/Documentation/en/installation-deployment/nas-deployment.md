# NAS Deployment

## What this page solves

- It gives NAS users one stable landing page instead of scattering the same setup across multiple platform pages.

## Recommended NAS platforms

- Synology
- QNAP
- Feiniu OS
- BaoTa
- Other NAS panels that support Docker or Compose

## Writing rule

- Explain shared preparation first: image, ports, volumes, environment variables.
- Then explain platform-specific differences.
- Do not turn panel differences into product differences.

## Completion standard

- NAS users can find their platform path from one page.
