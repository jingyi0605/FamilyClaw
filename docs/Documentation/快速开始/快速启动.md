# 快速启动

## 这页要解决什么

- 给第一次上手的人一条最短路径

## 最短路径

1. 先决定你要走哪条路：
   - 本地快速体验：优先看 [Docker安装](../安装部署/Docker安装.md)
   - 想自己改代码：优先看 [环境准备](../开发文档/环境准备.md)
   - 想先了解产品页面：优先看 [仪表盘](../使用指南/仪表盘.md)
2. 按对应页面完成第一次启动
3. 打开仪表盘，确认系统能正常进入主界面

## 启动后第一件事

- 先确认家庭和成员是否已经准备好
- 再去看设置页，把基础配置补齐

## 完成标准

- 读者知道第一步该看哪一页，而不是在目录里乱撞
