# 快速启动

## 这页要解决什么

- 给第一次上手的人一条最短路径，直接进入可用的网页端
- 告诉新人先做哪一步、用什么命令、成功后应该看到什么

## 请选择你的路径

1. 只想最快跑起来（推荐）：走 [Docker安装](../安装部署/Docker安装.md)，一条 `docker run` 就能起服务。
2. 需要二开或本地调试：按 [源码安装](../安装部署/源码安装.md) 搭建后端 + 前端。
3. 想先看功能界面：直接看 [仪表盘](../使用指南/仪表盘.md) 的截图说明，确认有哪些页面。

## 最短可运行路径（Docker）

1. 准备一台能装 Docker 的机器，至少 2GB 内存，端口 8080（网页）、4399（语音网关可选）。
2. 在终端执行：

```bash
docker run -d \
  --name familyclaw \
  -p 8080:8080 \
  -p 4399:4399 \
  -e FAMILYCLAW_DB_PASSWORD='change-me' \
  -e FAMILYCLAW_VOICE_GATEWAY_TOKEN='replace-me' \
  -v /srv/familyclaw-data:/data \
  jingyi0605/familyclaw:0.1.0
```

3. 约 1 分钟后浏览器打开 `http://服务器IP:8080`，能看到登录页即为成功。
4. 初始账号：`user` / `user`，登录后请立即修改密码。

## 成功后的第一件事

- 在设置页补齐时区、语言、默认 AI 供应商。
- 在家庭设置里确认至少有一个成员；没有的话按页面提示完成初始化向导。

## 完成标准

- 能通过浏览器访问首页并成功登录
- 至少完成一次家庭初始化或看到“已完成初始化”的状态
