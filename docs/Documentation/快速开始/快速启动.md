# 快速启动

如果你现在只想一件事: 尽快开始用上 FamilyClaw，就从这里往下走。

你不需要先把所有文档看完，只要先选一条最适合自己的路径。

## 先选你的起步方式

1. 如果你只想先把系统跑起来，直接走 [Docker安装](../安装部署/Docker安装.md)，这是最省事的方式。
2. 如果你准备本地开发、调试或二次修改，去看 [源码安装](../安装部署/源码安装.md)。
3. 如果你还没决定要不要安装，想先看看页面长什么样，先去看 [仪表盘](../使用指南/仪表盘.md)。

## 最快开始的方法: Docker

如果你当前目标很简单，就是“先看到登录页、先把系统跑起来”，按下面做就行。

1. 准备一台能安装 Docker 的机器，至少预留 2GB 内存。
2. 确保 `8080` 端口可用；如果你后面还想接入语音相关能力，再额外保留 `4399` 端口。
3. 在终端执行下面的命令：

```bash
docker run -d \
  --name familyclaw \
  -p 8080:8080 \
  -p 4399:4399 \
  -v /srv/familyclaw-data:/data \
  jingyi0605/familyclaw:latest
```

这里默认写 `latest`，让安装文档保持稳定。只有在你要精确回滚、复现旧版本问题，或者明确锁定某个发布版时，才改成具体版本标签。

4. 第一次启动时，系统会自动生成需要的密钥信息，并写入 `/srv/familyclaw-data/runtime/secrets/`。如果你要自定义数据库密码，优先传 `FAMILYCLAW_DB_PASSWORD`；如果同时传了 `FAMILYCLAW_DATABASE_URL`，两边密码必须一致。
5. 等大约 1 分钟后，在浏览器打开 `http://服务器IP:8080`。如果你已经看到登录页，说明系统已经跑起来了。
6. 初始账号是 `user`，初始密码也是 `user`。第一次登录后，按页面引导继续完成初始化，并尽快改掉默认账号密码。

## 登录后先做什么

- 在设置里补齐时区、语言和默认 AI 服务，先把基础环境配好。
- 在家庭设置里确认至少有一个成员；如果还没有，就按页面提示完成初始化。
- 如果你后面想让家里其他人也一起用，现在就可以开始整理家庭成员和常用场景。

## 看到这些，就算已经成功开始

- 你能在浏览器里打开首页并成功登录。
- 你已经完成初始化，或者页面已经显示“已完成初始化”。
- 接下来你可以去 [使用指南](../使用指南/仪表盘.md) 继续熟悉页面和功能。
