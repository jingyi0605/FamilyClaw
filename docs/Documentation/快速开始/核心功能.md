# 核心功能

## 当前核心模块

- 仪表盘：收口首页信息、提醒、状态卡片和关键入口
- 家庭：管理家庭、成员、房间及权限边界
- 对话：承载用户与系统的主要交互入口
- 记忆：沉淀长期上下文、观察记录和可回溯内容
- 设置：统一管理系统行为、接入配置和运行参数

## 能力边界

- 宿主负责统一规则、统一入口、统一数据语义
- 插件负责第三方接入、映射和扩展能力
