# 核心功能

FamilyClaw 的重点，不是把很多功能堆在一起，而是把家庭里经常会反复遇到的事情，放到一个顺手、能继续记住上下文的地方。

## 你会用到的几个核心功能

- 仪表盘：把家庭里常看的信息放到首页，比如提醒、天气、健康情况和常用入口，方便你一进来就知道现在该处理什么。
- 家庭：把成员、房间和家庭关系整理清楚，后面的很多功能都会围绕这些信息展开。
- 对话：你可以直接通过聊天来提问、处理事情和获取帮助；如果接入了合适的设备，家里长辈也可以通过更自然的方式来使用。
- 记忆：把重要的偏好、事件和家庭背景慢慢记下来，下次再聊类似事情时，不用总是从头再说一遍。
- 设置：把账户、语言、时区和常用服务统一整理在一起，方便你按自己的习惯调整。
- 插件：按需要接入更多服务和能力，让 FamilyClaw 更贴近你家的实际使用方式。

## 这些功能能用在什么场景里

- 想照顾家里老人时，可以把提醒、日常沟通和服务入口放在一起，让使用门槛更低。
- 想沉淀家庭记忆时，可以把家人的偏好、重要事件和习惯慢慢记下来，后面继续用。
- 想围绕健康或教育做长期陪伴时，也可以把相关信息和服务逐步接进来，不用每次换一个新工具。
- 想让不懂技术的家人也能参与时，可以通过更熟悉的方式接入，让他们不用学复杂操作。
