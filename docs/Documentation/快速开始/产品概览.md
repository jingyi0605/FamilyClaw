# 产品概览

FamilyClaw 是一个围绕家庭生活来组织信息、对话和常用能力的系统。

你可以把它理解成一个面向家庭场景的统一入口：家里的人、常用信息、日常对话和扩展功能，都可以放在同一个地方继续使用。

## 你可以用它做什么

- 把家庭成员、房间和相关信息整理在一起，减少分散记录和来回切换。
- 通过对话方式处理日常问题，让常用操作更自然，不用总是去翻菜单。
- 保存长期有用的信息，比如偏好、事件和历史内容，后面需要时还能继续找到。
- 围绕老人关怀、家庭记忆、健康和教育这些长期场景，逐步把家庭服务沉淀下来。
- 按自己的需要接入更多能力，让系统更贴近你的使用习惯。

## 它更适合哪类人

- 想把家庭相关的信息、对话和常用工具放到一个地方统一管理的人。
- 不想每次都重新说明背景，希望系统能逐步记住家庭情况和使用习惯的人。
- 希望把老人关怀、家庭记忆、健康、教育这些事情慢慢接进同一个家庭系统里的人。
- 希望家里不懂技术的长辈也能轻松用上 AI，比如让外婆直接通过微信或智能音响就能愉快地获得帮助和服务的人。
- 需要后续继续扩展功能，而不是只满足于一个固定页面的人。
