# 产品概览

## 产品是干什么的

FamilyClaw = 家庭上下文 + 对话 + 长期记忆 + 插件生态 + 统一入口。

- 家庭是主角：家庭、成员、房间、权限是一套基础数据。
- 对话是主要交互：网页端入口已就位，语音网关可选接入。
- 记忆是核心资产：对话、事件、偏好沉淀成可检索的家庭记忆。
- 插件让能力扩展：AI 供应商、通讯渠道、主题包等都以插件形式接入。
- 文档同构：中文根路径，英文 `/en/`，信息架构一致。
