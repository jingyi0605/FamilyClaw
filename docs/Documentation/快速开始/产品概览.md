# 产品概览

## 产品是干什么的

FamilyClaw 的目标不是做一个花哨的 AI 壳子，而是把家庭、对话、记忆、插件能力和统一入口收口成一套能长期维护的家庭 AI 平台。

## 现在对外讲清楚的核心点

- 一个家庭一套上下文：家庭、成员、房间、权限是基础数据
- 一个统一工作台：仪表盘、对话、记忆、设置都围绕同一套数据工作
- 一个插件系统：第三方能力通过插件接入，不往宿主里硬塞特例
- 一套可发布文档：中文根路径、英文 `/en/`，同构维护
