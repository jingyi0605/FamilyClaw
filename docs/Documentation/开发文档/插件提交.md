# 插件提交

## 提交前最低要求

1. 插件目录和 `manifest.json` 已按规范完成
2. 入口路径能对上真实代码
3. 权限、风险等级、类型声明清楚
4. 有可读的 README
5. 有最小验证说明和依赖清单

## 最少要交什么

- 插件源码仓库地址
- 插件说明文档地址
- 注册表条目
- PR 说明

## 审核方会重点看什么

- 条目和 `manifest.json` 是否一致
- 仓库是否可访问
- README 是否写清楚用途、权限和验证方式
- 风险等级是否瞎报
