# 记忆

## 页面现状

- **状态**：H5 记忆页已可用，入口 `/pages/memories/index`，受登录与家庭上下文保护。
- **数据来源**：后端 `/api/v1/memories` 相关接口；记忆含修订历史、可见性与状态。

## 能做什么

- 按类型筛选：全部 / 事实 / 事件 / 偏好 / 关系。
- 搜索：支持关键字搜索标题与内容。
- 查看详情：点击左侧列表项查看右侧详情，包含所有修订记录。
- 状态与可见性：
  - 可见性：公开 / 家庭 / 私密 / 敏感。
  - 状态：有效 / 待确认 / 已失效 / 已删除。
- 修订与操作：
  - 纠错（Correct）：提交修订，生成新的版本。
  - 标记失效（Invalidate）：保留历史，当前版本失效。
  - 删除（Delete）：逻辑删除，历史保留。

> 配图占位：记忆列表+详情+修订历史截图

## 使用步骤

1. 打开记忆页，等待列表加载完成（右上角家庭上下文要正确）。
2. 通过顶部筛选切换类型，或输入关键词搜索。
3. 点击左侧某条记忆，右侧展示详情与修订历史：
   - “当前版本”展示标题、摘要、可见性、状态。
   - “修订历史”按时间倒序，展示变更字段与原因。
4. 若需修订：
   - 选择“纠错”填写新的标题/内容/可见性后提交。
   - 选择“标记失效”或“删除”时，需填写原因后提交。
5. 提交后列表会刷新；失败时请根据提示检查权限或网络。

## 权限与提示

- 当前登录成员需具备相应权限才可修订或删除，权限不足时会返回错误提示。
- “敏感”与“私密”可见性的记忆，非授权成员不可见或不可操作。
- 删除是逻辑删除，历史版本仍可见以便审计。

## 常见问题

- **列表为空**：新环境首次使用可能尚无记忆；正常对话与事件会逐步产生记忆。
- **修订失败/权限不足**：确认当前成员角色与家庭上下文是否正确，必要时联系管理员。
- **时间显示异常**：浏览器时区与家庭时区不一致时会出现偏差，先在“设置”页调整时区。

## 完成标准

- 用户能筛选、搜索、查看、修订和删除记忆。
- 清楚可见性和状态的含义，并知道权限不足时的处理方式。
