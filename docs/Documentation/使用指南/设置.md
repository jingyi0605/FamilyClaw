# 设置

## 页面现状

- **状态**：H5 设置页已可用，入口 `/pages/settings/index`（受登录与家庭上下文保护）。
- **结构**：顶部导航 + 多个分区卡片；部分设置会跳转到子页（如 AI 设置）。

![fc-doc-20260320T000409.webp](assets/fc-doc-20260320T000409.webp)

## 已实现的主要分区

- **外观与主题**：选择内置或插件主题；若主题不可用会显示禁用提示，选择后立即生效并保存。
- **语言与时区**：选择家庭默认语言与时区，影响时间显示与翻译资源；变更后需等待界面刷新。
- **通知与可访问性**：当前以占位提示为主，后续版本会补充。
- **AI 设置快捷入口**：跳转到 AI 设置子页（见下方）。

## AI 设置子页（`/pages/settings/ai/index`）

- **AI 助手**：创建/编辑家庭管家或其他角色；需先完成模型供应商配置。
- **模型服务（Provider Profiles）**：基于插件适配的供应商表单，填写 `base_url`、`secret_ref`、`model_name` 等字段，支持启用/禁用。
- **能力路由（Routes）**：为 `text`、`qa_generation` 等能力选择主模型与备用模型，配置超时与重试。
- **初始化进度提示**：如果初始化未完成，这里会提示剩余步骤。

### 本地模型供应商

现在内置支持这些本地模型供应商插件：

- `Ollama`
- `LM Studio`
- `LocalAI`

另外还有十几种市面上主流的公有云AI服务提供商，支持多家服务商的Coding Plan。

最省事的配置顺序是：

1. 先选供应商插件
2. 填本地服务地址
3. 如果你的本地服务启用了 API Key，再把 API Key 填上
4. 页面会自动刷新模型列表
5. 直接从列表里选模型；如果你要填自定义模型名，也可以继续手动输入

这三个插件都会按“仅本地”处理，不会默认落到远端模型隐私级别。

## 家庭与成员上下文

- 页面顶部会显示当前家庭；切换家庭后，语言/主题/时区等设置随之切换。
- 如果家庭未初始化或被禁用，设置页会提示并引导回初始化向导。

## 常见问题

- **主题缺失/失效**：可能对应主题插件被禁用或卸载，选择其他可用主题即可。
- **语言/时区未生效**：刷新页面或重新登录；确认家庭上下文是否切到目标家庭。
- **无法保存 AI 供应商配置**：检查必填项与密钥变量是否正确，或确认对应插件已启用。

## 完成标准

- 用户能切换主题、语言、时区，并能进入 AI 设置完成模型与路由配置。
- 遇到不可用主题或未初始化家庭时，能根据页面提示采取正确操作。
