## 家庭创建与初始化

- 初始化向导完成后，系统会在你第一次进入应用时自动带你走一遍核心页面
- 这轮导览默认覆盖：
  - 首页
  - 家庭
  - 对话
  - 记忆
  - 设置
- 如果你中途刷新页面或切到别的页面，再回到应用时，系统会优先恢复到最近一步；如果某个目标区域刚好没显示出来，会自动退回当前页说明，不会把你卡死
- 如果你点击跳过，系统会把这一版导览记成“已处理”，后续不会反复打扰
- 如果你之后想重新看一遍，可以去设置页里的“重新查看导览”

## 成员关系图谱

- 家庭页里的成员关系图谱支持拖拽节点、缩放视图和点击成员切换关系视角
- 图谱卡片本身为了避免拖拽时误选文字，默认不支持复制
- 除了这块图谱卡片，家庭页和其他页面里的说明文字、列表文字、表单文本都应该可以正常选中和复制

## 常见问题

- 为什么历史用户不会每次进首页都被导览打断？
  - 因为导览完成状态是按成员记录的，只在当前成员还没完成当前版本时才会自动触发
- 跳过和完成有什么区别？
  - 对当前版本导览来说，两者都会结束本轮并记成“已处理”；区别只是你有没有把整套说明走完
