# 仪表盘

## 页面现状

- **状态**：H5 仪表盘已可用，需先登录并完成初始化向导（创建家庭、首位成员、AI 供应商、首个管家）。
- **入口**：登录后自动跳转 `/pages/home/index`，未登录会被 Guard 重定向到登录或初始化向导。

## 能看到什么

- 家庭欢迎区：显示当前家庭与成员昵称。
- 卡片区：内置卡片（天气、家庭统计、房间、成员、事件、快捷操作等），部分卡片由插件/集成提供。
- 卡片状态：`正常`、`空数据`、`过期`、`错误` 四种状态，错误时会显示原因。
- 快捷操作：跳转到对话、记忆、设置、家庭等入口（若目标页仍在迁移，会出现占位提示）。

## 怎么用

1. 打开仪表盘后先确认右上角已选中的家庭与成员，确保在正确的上下文。
2. 查看卡片：
   - 天气卡：来自天气插件，显示当前条件与温湿度，状态异常时优先检查插件启用与地区配置。
   - 统计卡：显示成员/房间/设备/提醒数量，数据来源于后端 `/dashboard/home`。
   - 事件/提醒卡：显示最近待办或日程。
   - 快捷操作卡：一键跳转对话、记忆、设置等核心入口。
3. 调整布局：
   - 拖拽卡片左上角“六点”可重新排序。
   - 拖拽卡片右下角可在“半宽/全宽”与“紧凑/常规/高”之间切换。
   - 调整后会自动保存到当前成员布局。

> 配图占位：拖拽卡片调整宽度与高度的操作示意

## 常见问题

- **卡片长时间显示“加载中/错误”**：检查网络、家庭上下文是否正确，再检查对应插件或集成是否启用。
- **天气卡无数据**：确认已在初始化或设置里选择地区，并安装可用的天气插件。
- **布局没保存**：浏览器需允许本地存储；重新登录后会从服务器恢复上次布局。

## 完成标准

- 首次登录的用户能读懂每张卡片的用途。
- 知道如何拖拽与调整卡片尺寸，并确认布局会被保存。
