## 关键注意事项

- H5 登录页只会在第一次使用 bootstrap 账号时默认填入 `user`。如果你已经把初始账号改成自己的正式账号，后续登录页不会再把 `user` 塞回来。
- 如果浏览器提示保存账号或密码，建议允许保存。下次回到登录页时，浏览器会直接帮你补全，省得每次重输。
