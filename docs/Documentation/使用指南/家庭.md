# 家庭

## 页面现状

- **状态**：H5 家庭页已可用，功能比较完整。
- **入口**：`/pages/family/index`
- **访问条件**：需要先登录，并且已经选定当前家庭。
- **页面结构**：顶部是家庭页头，下面有 5 个主标签页：
  - `概览`
  - `房间`
  - `设备`
  - `成员`
  - `关系`

这页不是单纯展示页。它会同时读取家庭资料、上下文概览、房间、设备、成员、成员偏好和成员关系，并允许直接修改其中一部分数据。

> 配图占位：家庭页整体截图  

## 你能在这里做什么

### 1. 管理家庭基础资料

在“概览”标签页里，可以查看和修改：

- 家庭名称
- 时区
- 默认语言
- 地区信息（国家 / 省 / 市 / 区）

同时还能看到当前家庭上下文摘要，例如：

- 家庭当前模式
- 隐私模式
- 是否启用了访客模式、儿童保护、长者关怀、语音快速通道等

如果你刚完成初始化，这里就是后续补资料和修资料的主入口。

> 配图占位：家庭概览与资料编辑区域

### 2. 新增房间并查看房间分布

在“房间”标签页里，可以：

- 查看现有房间列表
- 新建房间
- 为房间选择类型
- 设置房间隐私级别

当前创建房间时需要填写：

- 房间名称
- 房间类型
- 隐私级别

页面会把房间下的设备数量、活跃状态一起展示出来，方便你先判断家里结构是不是已经录完整。

目前“编辑房间”按钮还没有真正开放编辑动作，新增已经可用，列表展示也可用。

> 配图占位：房间列表与新建房间弹窗

### 3. 查看设备并按条件筛选

在“设备”标签页里，可以：

- 查看当前家庭下的设备列表
- 按房间筛选
- 按设备类型筛选
- 按设备状态筛选
- 打开设备详情弹窗

设备页会展示的信息包括：

- 设备名称
- 所属房间
- 设备类型
- 当前运行状态
- 是否可控制
- 是否启用

如果某个设备来自集成插件，这一页通常能最快帮你看出：

- 设备有没有同步进来
- 同步进来了但状态是不是异常
- 房间绑定是不是对的

> 配图占位：设备筛选区与设备详情弹窗

### 4. 新增成员、编辑成员、停用成员

在“成员”标签页里，可以直接做成员管理，这是当前家庭页里最完整的一块之一。

已支持的动作包括：

- 新增成员
- 修改成员资料
- 设置昵称、性别、角色、年龄段、生日、手机号
- 为儿童指定监护人
- 启用 / 停用成员
- 编辑成员偏好

当前成员角色包括：

- 管理员
- 成人
- 儿童
- 长者
- 访客

页面还会结合生日、农历生日标记、监护人信息和成员偏好，给出更完整的成员卡片信息。

如果你要维护家庭成员资料，这一页基本就是主工作台。

> 配图占位：成员列表  
> 配图占位：新增成员弹窗  
> 配图占位：成员偏好编辑区域

### 5. 维护成员关系

在“关系”标签页里，可以：

- 查看现有成员关系
- 用图形方式查看关系网络
- 新增关系
- 删除关系

当前关系管理是按“源成员 -> 关系类型 -> 目标成员”来创建的。  
这对后续的称呼理解、照护关系、家庭协作都很重要。

如果家里有儿童、长者或者需要明确监护 / 照护关系，这一页最好认真补完整。

## 页面里的几个真实边界

为了避免你被文档误导，这里把当前边界说清楚：

- H5 家庭页是可用的，不是占位页。
- 房间支持新增和查看，但当前“编辑房间”按钮还是禁用态。
- 设备页主要是查看、筛选和打开详情，不是全量设备控制中心。
- 成员管理已经支持新增、编辑、状态切换和偏好保存。
- 成员关系支持新增和删除，也有关系图展示。
- 页面如果局部接口失败，会显示“部分加载失败”提示，但不会把整个家庭页直接打死。

## 推荐使用顺序

如果你是第一次进入家庭页，推荐按这个顺序补齐：

1. 先看“概览”，确认家庭名称、时区、语言、地区对不对。
2. 再去“房间”，把家里的主要空间补齐。
3. 接着看“成员”，把家人资料补完整。
4. 再去“关系”，把监护、父母、配偶、子女等关系补清楚。
5. 最后看“设备”，确认集成过来的设备是否已经落到正确房间。

## 常见问题

### 家庭页打开后有些卡片空白

这通常不是“家庭页没做好”，而是某部分数据还没录，或者对应接口拉取失败了。先看页面顶部有没有“部分加载失败”提示。

### 为什么设备页里有设备，但房间里数量不对

先检查设备是不是已经绑定到正确房间。家庭页展示的是当前后端返回的数据，不会替你脑补映射关系。

### 儿童成员为什么必须选监护人

这是页面当前的表单规则，不是文档要求。儿童角色需要补监护关系，避免家庭结构数据残缺。

### 我改完成员资料，为什么别的页面没立刻变化

家庭页会刷新自己的工作区数据，但其他页面是否立即同步，还取决于那些页面自己的刷新时机。

## 完成标准

- 你知道家庭页已经是可用主页面，而不是迁移占位。
- 你知道这页现在至少能管理家庭资料、房间、设备视图、成员和关系。
- 你也清楚当前仍未完全打通的地方主要在哪，不会把“局部未做完”误判成“整个页面没实现”。
