# 家庭

## 页面现状

- **状态**：当前 H5 “家庭”页仍是迁移占位页（显示占位提示，暂不提供操作）。
- **可用功能**：无交互，仅提示迁移状态。
- **不可用功能**：在该页直接新增/编辑家庭、成员、房间、权限。

## 现阶段能做什么

- **初始化向导**：首次登录时，在“初始化”步骤完成家庭信息、首位成员资料与时区/地区设置。
- **后台 & 设置页**：已创建的家庭、成员、地区、时区可在“设置”页（语言与时区部分）查看与调整；更深入的成员管理需通过后台 API（`/households`、`/members` 等）。

## 等待补齐的能力

- H5 家庭页将提供：家庭资料编辑、成员/房间管理、地区选择、权限与角色可视化。
- 目前如需变更成员/房间，请走初始化向导或后端接口，避免直接改数据库。

## 建议操作路径

1. 新环境：按“首次登录与初始化”向导完成家庭与首位成员创建。
2. 已运行环境：在“设置”页调整家庭语言/时区；如需成员与房间维护，暂用后台接口。
3. 遇到家庭页占位：属正常现状，无需排障。

## 完成标准

- 用户知道家庭页暂不可操作，不会在此等待功能。
- 知道当前可行的替代路径（初始化向导或后台接口）。
