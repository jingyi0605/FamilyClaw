# 对话 / 助手

## 页面现状

- **状态**：当前 H5 “助手/对话”页仍是迁移占位页（只显示迁移提示）。
- **可用功能**：无交互；点击入口只会看到占位信息。
- **不可用功能**：在 H5 内发起/查看对话、回溯上下文、执行插件。

## 现阶段可用的对话方式

- 通过已启用的通讯渠道插件（如飞书、钉钉、Telegram、WeCom 等）与家庭 AI 交互。
- 如需 Web 端对话：等待新页落地前，可使用后端 WebSocket 接口 `/api/v1/realtime/conversation` 进行调试。

## 何时使用占位页

- 看到占位提示说明功能尚未上线，不需要重复刷新或排查。
- 如果需要在浏览器完成对话，请先确认是否有可用的外部渠道或 CLI/调试工具。

## 未来将补齐

- H5 会提供完整对话流：会话列表、实时流式响应、对话内插件提案与确认、上下文回溯。
- 日后上线后，快捷入口（仪表盘快捷操作）将直接跳转到可用的对话页。

## 完成标准

- 用户清楚当前页面不可用，避免在此排障。
- 知道可用的替代交互方式：启用渠道插件或通过后端接口调试。
