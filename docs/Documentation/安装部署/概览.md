# 安装部署概览

## 这页要解决什么

- 帮你判断应该走哪一种安装或部署路径

## 路径选择

- 最快体验（推荐）：看 [Docker安装](./Docker安装.md)，一条命令即可。
- 要二开或调试：看 [源码安装](./源码安装.md)。
- NAS 面板：看 [NAS部署](./NAS部署.md)。
- 纯 Linux 服务器：看 [Ubuntu部署](./Ubuntu部署.md)。
- Windows 本地演示：看 [Windows部署](./Windows部署.md)。

## 选择原则

- 能用 Docker 先用 Docker；需要改代码再切源码。
- 所有方案都共用同一产品逻辑：网页端 8080、语音网关 4399、数据目录 `/data`。
- 如果你只需要网页对话，语音网关端口可以不暴露。

## 完成标准

- 读者能判断自己应该打开哪一页，而不是每页都扫一遍
