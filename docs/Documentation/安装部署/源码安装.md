# 源码安装

## 适用场景

- 需要修改代码、调试或跑分支。
- 拥有本机或服务器的 Python 3.11、Node.js 20 环境，以及外部 PostgreSQL。

## 环境要求

- Python 3.11+
- Node.js 20+（npm 已自带）
- PostgreSQL 15+，创建数据库 `familyclaw`、用户 `familyclaw`，密码自定。
- 操作系统：macOS / Linux / WSL / Windows 均可。

## 步骤 1：拉取代码

```bash
git clone https://github.com/jingyi0605/FamilyClaw.git
cd familyclaw
```

## 步骤 2：准备后端配置

```bash
cp apps/api-server/.env.example apps/api-server/.env
```

至少按你本地数据库修改这些项（文件中已给默认值）：

- `FAMILYCLAW_DATABASE_URL`：如 `postgresql+psycopg://familyclaw:密码@127.0.0.1:5432/familyclaw`
- `FAMILYCLAW_TEST_DATABASE_URL`：测试库连接
- `OPENAI_API_KEY`：如需调用 OpenAI 再填

## 步骤 3：启动后端（API）

```bash
bash apps/start-api-server.sh
```

脚本会：

- 检查 Python 3.11，创建虚拟环境 `apps/api-server/.venv`
- 安装依赖（基于 `pyproject.toml`）
- 自动跑 Alembic migration
- 启动 Uvicorn 开发服（默认 `0.0.0.0:8000`，可通过 `HOST`/`PORT` 环境变量覆盖）

验证：浏览器或 curl 打开 `http://127.0.0.1:8000/api/v1/healthz` 返回 `{"status":"ok"}`。

## 步骤 4：启动前端（H5）

```bash
npm install --legacy-peer-deps
npm --prefix ./apps/user-app install --legacy-peer-deps
npm run dev:user-app:h5
```

- `apps/user-app/config/platform/h5.ts` 已配置 `/api` 代理到 `http://127.0.0.1:8000`。
- H5 开发地址以终端输出为准，不要死记某个端口。

验证：浏览器访问终端给出的 H5 地址，能看到登录页。


## 启动后登录

- 初始账号：`user` / `user`（登录后请改密码）。
- 若提示初始化向导，按页面要求补家庭信息即可。

## 常见问题

- 后端连不上数据库：确认 PostgreSQL 运行、账号密码正确、端口未被占用。
- 前端 10086 访问不到：查看终端日志，确认 Taro dev server 未报错；或端口被占用。
- 跨域或接口 404：确认前端代理到 `http://127.0.0.1:8000`，后端保持运行。
- Migration 报错：先确保数据库空库；必要时清空后重新跑 `bash apps/start-api-server.sh`。
