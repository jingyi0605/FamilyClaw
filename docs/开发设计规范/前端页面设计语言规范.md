## 页面布局

### 最大宽度

所有页面统一使用 **1440px** 最大宽度，居中显示：

```scss
.page--your-page {
  --page-content-max-width: 1440px;
  max-width: var(--page-content-max-width);
  margin: 0 auto;
}
```

### 页面结构

标准页面结构包含两个主要部分：

```tsx
<div className="page page--your-page">
  <PageHeader
    title={t('yourPage.title')}
    actionsClassName="page-header__actions--tabs"
    actions={(
      <div className="memory-main-tabs">
        {/* 标签页按钮 */}
      </div>
    )}
  />

  {/* 页面内容 */}
  <div className="your-page-content">
    {/* 具体内容 */}
  </div>
</div>
```

### 全局样式文件

在 `apps/user-app/src/runtime/h5-shell/styles/index.h5.scss` 中为每个页面定义基础变量：

```scss
.page--your-page {
  /* 页面布局 */
  --page-content-max-width: 1440px;

  /* 卡片尺寸 */
  --card-min-height-compact: 160px;
  --card-min-height-regular: 200px;
  --card-min-height-tall: 280px;
  --card-padding: var(--spacing-md);
  --card-gap: var(--spacing-md);
  --card-radius: var(--radius-lg);

  /* 图标尺寸层级 */
  --icon-size-xs: 16px;
  --icon-size-sm: 18px;
  --icon-size-md: 20px;
  --icon-size-lg: 32px;
  --icon-size-xl: 40px;
}
```

## 标签页设计

### 统一标签页样式

所有页面使用 `.memory-main-tabs` 和 `.memory-main-tab` 样式类。默认情况下，标签栏应直接挂在 `PageHeader.actions` 中，和页面标题合并成同一个头部区域，但标签本身保持左对齐，不要再为家庭页、设置页单独造一套 `family-tab`、`settings-tab` 之类的私有样式：

```tsx
<PageHeader
  title={t('yourPage.title')}
  actionsClassName="page-header__actions--tabs"
  actions={(
    <div className="memory-main-tabs">
      <button
        type="button"
        className={`memory-main-tab ${activeTab === 'tab1' ? 'memory-main-tab--active' : ''}`}
        onClick={() => setActiveTab('tab1')}
      >
        {t('yourPage.tab.tab1')}
      </button>
      <button
        type="button"
        className={`memory-main-tab ${activeTab === 'tab2' ? 'memory-main-tab--active' : ''}`}
        onClick={() => setActiveTab('tab2')}
      >
        {t('yourPage.tab.tab2')}
      </button>
    </div>
  )}
/>
```

如果标签栏属于卡片或区块内部，而不是页面主标题，则把同样的 `.memory-main-tabs` 放到对应区块标题的 `actions` 槽位里，原则不变：样式统一、标签左对齐、不要单独造新皮肤。

### 标签页样式定义

在全局样式文件中已定义：

```scss
.memory-main-tabs {
  display: flex;
  gap: var(--spacing-xs);
  margin-bottom: var(--spacing-lg);
  border-bottom: 1px solid var(--border-light);
}

.memory-main-tab {
  appearance: none;
  background: transparent;
  border: none;
  padding: var(--spacing-sm) var(--spacing-lg);
  font-size: var(--font-size-md);
  font-weight: 500;
  line-height: 1.4;
  color: var(--text-secondary);
  border-bottom: 2px solid transparent;
  cursor: pointer;
  transition: all var(--transition);
}

.memory-main-tab:hover {
  color: var(--text-primary);
  background: var(--bg-card-hover);
}

.memory-main-tab--active {
  color: var(--brand-primary);
  border-bottom-color: var(--brand-primary);
}
```

### 移动端适配

在页面级样式文件中添加滚动支持：

```scss
.page--your-page {
  .memory-main-tabs {
    overflow-x: auto;
    scrollbar-width: none;
    &::-webkit-scrollbar { display: none; }
  }

  .memory-main-tab {
    white-space: nowrap;
    flex-shrink: 0;
  }
}
```

## 样式变量

### 推荐使用的主题变量

| 变量名 | 用途 |
|--------|------|
| `--bg-app` | 应用背景色 |
| `--bg-card` | 卡片背景色 |
| `--bg-card-hover` | 卡片悬停背景色 |
| `--bg-input` | 输入框背景色 |
| `--bg-surface` | 表面背景色 |
| `--text-primary` | 主要文字色 |
| `--text-secondary` | 次要文字色 |
| `--text-tertiary` | 三级文字色 |
| `--border` | 边框色 |
| `--border-light` | 浅边框色 |
| `--brand-primary` | 品牌主色 |
| `--brand-primary-light` | 品牌主色浅色 |
| `--spacing-xs` ~ `--spacing-xxl` | 间距层级 |
| `--radius-sm` ~ `--radius-xl` | 圆角层级 |
| `--shadow-sm` ~ `--shadow-lg` | 阴影层级 |
| `--transition` | 过渡时间 |

### 间距层级

```scss
--spacing-xs: 4px;
--spacing-sm: 8px;
--spacing-md: 12px;
--spacing-lg: 16px;
--spacing-xl: 24px;
--spacing-xxl: 32px;
```

### 字体层级

```scss
--font-size-xs: 12px;
--font-size-sm: 14px;
--font-size-md: 16px;
--font-size-lg: 18px;
--font-size-xl: 20px;
--font-size-xxl: 24px;
```

### 按钮变量

**重要**：按钮相关变量使用 `px` 单位定义，避免 Taro 的 `rem` 转换问题。

| 变量名 | 值 | 用途 |
|--------|-----|------|
| `--btn-gap` | `8px` | 按钮内图标与文字间距 |
| `--btn-padding-xs` | `4px 8px` | 超小按钮内边距 |
| `--btn-padding-sm` | `6px 14px` | 小按钮内边距 |
| `--btn-padding-md` | `10px 20px` | 中等按钮内边距（默认） |
| `--btn-padding-lg` | `12px 28px` | 大按钮内边距 |
| `--btn-radius` | `var(--radius-lg)` | 按钮圆角 |
| `--btn-shadow` | `var(--shadow-sm)` | 按钮阴影 |
| `--btn-shadow-hover` | `var(--shadow-md)` | 按钮悬停阴影 |
| `--btn-font-weight` | `600` | 按钮字重 |
| `--btn-min-width` | `80px` | 按钮最小宽度 |
| `--btn-min-width-primary` | `100px` | 主按钮最小宽度 |

#### 按钮使用示例

```tsx
// 主要按钮
<button className="btn btn--primary btn--sm">确认</button>

// 次要按钮
<button className="btn btn--outline btn--sm">取消</button>

// 默认尺寸
<button className="btn btn--primary">保存</button>
```

#### 按钮样式效果

- **主按钮 (`.btn--primary`)**：品牌色背景 + 白色文字 + 阴影 + 悬停上浮效果
- **次级按钮 (`.btn--outline`)**：透明背景 + 边框 + 悬停高亮
- **幽灵按钮 (`.btn--ghost`)**：卡片背景 + 浅边框

## 禁止使用的样式

### 禁止使用的效果

```scss
// ❌ 禁止：模糊滤镜
backdrop-filter: blur(18px);

// ❌ 禁止：复杂渐变背景
background: linear-gradient(180deg, color-mix(...), ...);

// ❌ 禁止：color-mix 阴影
box-shadow: 0 18px 40px color-mix(in srgb, var(--text-primary) 7%, transparent);

// ❌ 禁止：::before 装饰蒙版
.page--your-page::before {
  content: '';
  position: absolute;
  /* ... */
}
```

### 推荐的替代方案

```scss
// ✅ 推荐：使用纯色背景
background: var(--bg-card);

// ✅ 推荐：使用预设阴影
box-shadow: var(--shadow-sm);

// ✅ 推荐：使用简单的主题变量边框
border: 1px solid var(--border-light);
```

## 文件结构规范

### 页面文件结构

```
apps/user-app/src/pages/your-page/
├── index.h5.tsx          # 页面组件
├── index.h5.scss         # 页面样式
├── page.shared.ts        # 共享逻辑（可选）
└── components/           # 页面级组件（可选）
    └── YourComponent.tsx
```

### 样式文件命名

- 页面样式：`index.h5.scss`
- 组件样式：内联或与组件同名的 `.scss` 文件

### 导入顺序

```tsx
// 1. React 相关
import { useState, useEffect } from 'react';

// 2. Taro 相关
import Taro from '@tarojs/taro';

// 3. UI 组件

// 5. 本地模块

// 6. 样式文件

```

## i18n 翻译规范

### 翻译键命名规范

```typescript
// 页面标题
'yourPage.title': '页面标题',

// 标签页
'yourPage.tab.tab1': '标签一',
'yourPage.tab.tab2': '标签二',
'yourPage.tab.addTab': '添加标签页',

// 功能开发中
'yourPage.tab.comingSoonTitle': '功能开发中',
'yourPage.tab.comingSoonDesc': '该功能正在紧张开发中，敬请期待！',
```

### 翻译文件位置

- 中文：`apps/user-app/src/runtime/h5-shell/i18n/I18nProvider.tsx` 中的 `SHELL_MESSAGES['zh-CN']`
- 英文：`apps/user-app/src/runtime/h5-shell/i18n/I18nProvider.tsx` 中的 `SHELL_MESSAGES['en-US']`

### 页面级翻译

如果翻译内容较多，可以添加到：
- `apps/user-app/src/runtime/h5-shell/i18n/pageMessages.zh-CN.ts`
- `apps/user-app/src/runtime/h5-shell/i18n/pageMessages.en-US.ts`

## 示例代码

### 完整页面模板

```tsx
// index.h5.tsx
import { useState } from 'react';
import Taro from '@tarojs/taro';
import { EmptyStateCard, PageHeader, UiCard } from '@familyclaw/user-ui';
import { Construction, Home, Settings } from 'lucide-react';
import { useI18n } from '../../runtime/h5-shell';
import type { ShellMessageKey } from '../../runtime/h5-shell/i18n/I18nProvider';
import './index.h5.scss';

type TabKey = 'home' | 'settings' | 'comingSoon';

export default function YourPage() {
  const { t } = useI18n();
  const [activeTab, setActiveTab] = useState<TabKey>('home');

  const tabs = [
    { key: 'home' as const, labelKey: 'yourPage.tab.home', available: true },
    { key: 'settings' as const, labelKey: 'yourPage.tab.settings', available: true },
    { key: 'comingSoon' as const, labelKey: 'yourPage.tab.comingSoon', available: false },
  ];

  function renderContent() {
    switch (activeTab) {
      case 'home':
        return (
          <div className="your-page-grid">
            {/* 内容卡片 */}
          </div>
        );
      case 'settings':
        return (
          <div className="your-page-settings">
            {/* 设置内容 */}
          </div>
        );
      case 'comingSoon':
        return (
          <div className="your-page-coming-soon">
            <EmptyStateCard
              icon={<Construction size={48} className="text-text-tertiary" />}
              title={t('yourPage.tab.comingSoonTitle')}
              description={t('yourPage.tab.comingSoonDesc')}
            />
          </div>
        );
    }
  }

  return (
    <div className="page page--your-page">
      <PageHeader title={t('yourPage.title')} />

      <div className="memory-main-tabs">
        {tabs.map(tab => (
          <button
            key={tab.key}
            type="button"
            className={`memory-main-tab ${activeTab === tab.key ? 'memory-main-tab--active' : ''}`}
            onClick={() => setActiveTab(tab.key)}
          >
            {t(tab.labelKey)}
          </button>
        ))}
      </div>

      {renderContent()}
    </div>
  );
}
```

## 检查清单

新增页面或标签页时，请确认：

- [ ] 页面最大宽度设置为 1440px
- [ ] 使用 `PageHeader` 组件作为页面标题
- [ ] 标签页使用 `.memory-main-tabs` 和 `.memory-main-tab` 样式
- [ ] 没有使用 `backdrop-filter`
- [ ] 没有使用复杂 `linear-gradient`
- [ ] 没有使用 `color-mix` 阴影
- [ ] 没有使用 `::before` 装饰蒙版
- [ ] 使用主题变量（`var(--bg-card)` 等）
- [ ] 添加了 i18n 翻译键
- [ ] 移动端标签页支持水平滚动
